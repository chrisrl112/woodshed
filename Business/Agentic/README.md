# Agentic

AI-optimization and AI-ops research for how you *run* the business with AI — not the product's own AI features (those live in `pipelines/`, e.g. the OMR/transcription pipeline, the feedback agent, the janitor agent — see `docs/WOODSHED-FEEDBACK-AGENT-README.md`, `docs/WOODSHED-JANITOR-SPEC.md`).

Scope: prompt caching strategy, model selection/tradeoffs, agent oversight and project management, workflow/tooling research for getting more leverage out of AI across Branding, Features, Design, Social, and Strategy.
